# GoalPace Frontend

Фронтенд часть приложения GoalPace.

## Технологии

- React 18
- Vite
- Tailwind CSS
- React Router

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на http://localhost:5173

## Структура

```
src/
├── api/          # Функции для работы с API
├── components/   # React компоненты
├── pages/        # Страницы приложения
└── App.jsx       # Главный компонент
```
